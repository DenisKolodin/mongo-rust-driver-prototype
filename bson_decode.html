Implementation for BsonParser<T> where <T: Stream<u8>>

Method document

fn document(&mut self) -> Result<BsonDocument, ~str>

Parse a byte stream into a BsonDocument. Returns an error string on parse failure. Initializing a BsonParser and calling document() will fully convert a ~[u8] into a BsonDocument if it was formatted correctly.

Method new

fn new(stream: T) -> BsonParser<T>

Create a new parser with a given stream.

Function decode

fn decode(b: ~[u8]) -> Result<BsonDocument, ~str>

Standalone decode binding. This is equivalent to initializing a parser and calling document().
